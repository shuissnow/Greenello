# スコープ定義書

## 含むもの

- ユーザー認証（新規登録・ログイン・ログアウト・パスワード変更）
- タスクの CRUD（作成・参照・更新・削除）
- タスクの進捗ステータス管理（Drag & Drop 対応）
- 複数の表示形式（カンバン・リスト・カレンダー・進捗サマリー）
- タスクの検索・フィルター（キーワード・期限・優先度・カテゴリ）
- タスクの CSV エクスポート
- タスクの親子関係管理（親→サブタスクの2階層のみ）
- タスクへの担当者アサイン
- ボードの削除
- メール通知（タスク期限切れ通知・ボード招待メール）

## 含まないもの

- ソーシャルログイン（OAuth）
- リアルタイム通知・WebSocket 連携
- Webhook 通知・外部ツール連携（Slack・Teams 等）
- ボードメンバーの除名
- 有料プラン・課金機能
- モバイルネイティブアプリ（iOS / Android）
