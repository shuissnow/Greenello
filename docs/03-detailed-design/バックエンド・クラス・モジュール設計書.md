## Controllers（レイヤー: HTTP 受付・レスポンス返却）

### AuthController

- **概要:** 認証・アカウント管理エンドポイントを担当
- **責務:** セッション操作、認証チェックの委譲
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | Register | RegisterRequest | 201 / 422 | AuthService.RegisterAsync を呼び出す |
  | Login | LoginRequest | 200 / 401 / 423 | AuthService.LoginAsync → セッション書き込み |
  | Logout | — | 204 | セッション破棄 |
  | ChangePassword | ChangePasswordRequest | 204 / 422 | AuthService.ChangePasswordAsync → 全セッション無効化 |
- **依存クラス/モジュール:** AuthService

### BoardsController

- **概要:** ボード関連エンドポイントを担当
- **責務:** ボード CRUD
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | GetBoards | — | 200 `Board[]` | BoardService.GetUserBoardsAsync |
  | CreateBoard | CreateBoardRequest | 201 `Board` | BoardService.CreateBoardAsync |
  | DeleteBoard | boardId | 204 / 403 / 404 | BoardService.DeleteBoardAsync（オーナー確認） |
- **依存クラス/モジュール:** BoardService

### TasksController

- **概要:** タスク・サブタスク CRUD・CSV エクスポートを担当
- **責務:** タスク操作の委譲、ボード参加チェックの委譲
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | GetTasks | boardId, TaskFilters | 200 `Task[]` | TaskService.GetTasksAsync |
  | CreateTask | boardId, CreateTaskRequest | 201 `Task` | TaskService.CreateTaskAsync |
  | ExportCsv | boardId, TaskFilters | 200 CSV / 400 | CsvExportService.ExportAsync |
  | GetTask | taskId | 200 `Task` / 404 | TaskService.GetTaskAsync |
  | UpdateTask | taskId, UpdateTaskRequest | 200 `Task` / 404 | TaskService.UpdateTaskAsync |
  | DeleteTask | taskId | 204 / 404 | TaskService.DeleteTaskAsync（サブタスク連鎖削除） |
  | CreateSubtask | taskId, CreateSubtaskRequest | 201 `Task` / 422 | TaskService.CreateSubtaskAsync（2階層制限） |
- **依存クラス/モジュール:** TaskService, CsvExportService


## Services（レイヤー: ビジネスロジック）

### AuthService

- **概要:** 認証・アカウント管理のビジネスロジック
- **責務:** パスワードハッシュ、ログイン失敗カウント、セッション管理
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | RegisterAsync | email, password | User | メールアドレス重複確認 → bcrypt ハッシュ → INSERT |
  | LoginAsync | email, password | User | ロック確認 → パスワード検証 → 失敗カウント更新 |
  | ChangePasswordAsync | userId, currentPw, newPw | void | 現在PW検証 → 新PW ハッシュ → UPDATE |
- **依存クラス/モジュール:** UserRepository

### BoardService

- **概要:** ボード管理のビジネスロジック
- **責務:** ボード所有権確認
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | GetUserBoardsAsync | userId | `Board[]` | オーナーのボード一覧取得 |
  | CreateBoardAsync | userId, name | Board | ボード INSERT（owner_user_id = userId） |
  | DeleteBoardAsync | boardId, userId | void | オーナー確認 → ボード削除（タスク連鎖削除） |
  | IsOwnerAsync | boardId, userId | bool | boards.owner_user_id と一致するか確認 |
- **依存クラス/モジュール:** BoardRepository

### TaskService

- **概要:** タスク・サブタスク CRUD のビジネスロジック
- **責務:** 2階層制限チェック、サブタスク連鎖削除
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | GetTasksAsync | boardId, filters | `Task[]` | フィルター条件付きタスク一覧取得 |
  | CreateTaskAsync | boardId, userId, input | Task | タスク INSERT (status=inbox) |
  | GetTaskAsync | taskId, userId | Task | タスク取得（ボード参加確認付き） |
  | UpdateTaskAsync | taskId, userId, input | Task | タスク部分更新 |
  | DeleteTaskAsync | taskId, userId | void | サブタスクを先に DELETE → 親タスク DELETE |
  | CreateSubtaskAsync | parentTaskId, userId, title | Task | 親タスクの parent_task_id が null か確認（2階層制限） → INSERT |
- **依存クラス/モジュール:** TaskRepository, BoardService

### NotificationService

- **概要:** 期限切れタスク通知メールのビジネスロジック
- **責務:** 送信対象抽出、重複送信防止、リトライ制御
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | SendOverdueNotificationsAsync | — | void | 期限切れタスク抽出 → 通知ログ確認 → メール送信 → ログ記録 |
- **依存クラス/モジュール:** TaskRepository, IEmailService, NotificationRepository

### CsvExportService

- **概要:** タスク一覧の CSV エクスポートビジネスロジック
- **責務:** フィルター適用、CSV 生成（UTF-8 BOM付き）
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | ExportAsync | boardId, userId, filters | `byte[]` | タスク取得 → 0件チェック → CSV 変換 |
- **依存クラス/モジュール:** TaskRepository, BoardService

## BackgroundServices

### NotificationHostedService

- **概要:** 期限切れタスク通知を毎日 08:00 JST に実行する Hosted Service
- **責務:** タイマー管理、NotificationService の定期呼び出し
- **メソッド一覧:**
  | メソッド名 | 引数 | 戻り値 | 処理概要 |
  |-----------|------|--------|---------|
  | ExecuteAsync | CancellationToken | Task | 次回 08:00 JST までの待機 → NotificationService.SendOverdueNotificationsAsync 呼び出し |
- **依存クラス/モジュール:** NotificationService
- **タイマー計算（擬似コード）:**
  ```
  // 実行時刻は環境変数 NOTIFICATION_JOB_HOUR_JST（デフォルト: 8）で設定
  jstZone = TimeZoneInfo.FindSystemTimeZoneById("Tokyo Standard Time")
  while (!cancellationToken.IsCancellationRequested):
    nowJst = TimeZoneInfo.ConvertTimeFromUtc(DateTime.UtcNow, jstZone)
    nextRunJst = nowJst.Date + TimeSpan.FromHours(NOTIFICATION_JOB_HOUR_JST)
    if nowJst >= nextRunJst:
      nextRunJst += 1日   // 当日08:00を過ぎていたら翌日08:00を目標にする
    waitSeconds = (nextRunJst - nowJst).TotalSeconds
    await Task.Delay(waitSeconds, cancellationToken)
    await _notificationService.SendOverdueNotificationsAsync()
  ```

## Middleware

### ErrorHandlingMiddleware

- **概要:** 全ての未ハンドリング例外をキャッチし、統一エラー JSON 形式で返す
- **責務:** スタックトレース非公開、エラーログ出力、500 レスポンス返却
- **依存クラス/モジュール:** ILogger

### AuthenticationMiddleware

- **概要:** 認証が必要なエンドポイントでセッション有効性を検証する
- **責務:** セッション確認 → 401 返却 / HttpContext.User 設定
- **依存クラス/モジュール:** ISession
