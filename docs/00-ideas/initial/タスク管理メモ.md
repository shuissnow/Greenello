# 	 タスク管理アプリのアイデアメモ

## 背景

技術的な学習を目的としたWebアプリを作成したい。
基本的なCRUD機能や、機能拡張をしやすいという点で Trello風のTodoアプリを作成したい。
複数人が同じTODOを管理できるようにしたい。

## やりたいこと

### タスク

- タスクの追加

    - 新規追加時はInboxとする。そのご、UserがDrag＆Dropによって進捗状況を変更可能
    - タイトルだけ入力可能だが、タスクのカテゴリや期限などの詳細な部分も入力可能

- タスクの表示が可能

    - タスクの表示形式を変更可能
        - カンバン形式
        - リスト形式
        - カレンダー形式
        - ダッシュボード（タスクの進捗具合をグラフ化）
    - タスクの親子間を分かりやすく表示する

- タスクの検索

    - キーワード、期限、優先度、カテゴリによるフィルターが可能

- タスクの編集

    - タスクのタイトル、優先度、カテゴリなどの個別で更新可能

- タスクの削除

    - タスクのチェックボックスの押下、またはDrag&Dropによって完了とする。
    - 完了可能したTodoのタイトルは打ち消し線を入れる

    

    

### 出力機能

- CSVとして出力可能
    - 出力可能なTODOをフィルター可能

### ログイン

- Sessinon管理を用いた認証機能
