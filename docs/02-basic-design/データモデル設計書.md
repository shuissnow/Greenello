## 各エンティティ定義

### users

| カラム名 | 型 | 制約 | 説明 |
|---------|---|------|------|
| id | UUID | PK, NOT NULL | ユーザーID |
| email | VARCHAR(255) | NOT NULL, UNIQUE | メールアドレス |
| password_hash | VARCHAR(255) | NOT NULL | bcrypt ハッシュ |
| failed_login_count | INT | NOT NULL, DEFAULT 0 | ログイン失敗回数 |
| locked_until | TIMESTAMP | NULL | アカウントロック解除日時（NULL=非ロック） |
| created_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 作成日時 |
| updated_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 更新日時 |

### boards

| カラム名 | 型 | 制約 | 説明 |
|---------|---|------|------|
| id | UUID | PK, NOT NULL | ボードID |
| name | VARCHAR(100) | NOT NULL | ボード名 |
| owner_user_id | UUID | FK(users.id), NOT NULL | ボードオーナーのユーザーID |
| created_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 作成日時 |
| updated_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 更新日時 |

### tasks

| カラム名 | 型 | 制約 | 説明 |
|---------|---|------|------|
| id | UUID | PK, NOT NULL | タスクID |
| board_id | UUID | FK(boards.id), NOT NULL | 所属ボードID |
| parent_task_id | UUID | FK(tasks.id), NULL | 親タスクID（NULL=親タスク） |
| title | VARCHAR(100) | NOT NULL | タスクタイトル |
| status | VARCHAR(20) | NOT NULL | `inbox` / `in_progress` / `done` |
| priority | VARCHAR(10) | NULL | `high` / `medium` / `low` / NULL（未設定） |
| category | VARCHAR(20) | NULL | カテゴリ（自由入力） |
| due_date | DATE | NULL | 期限日（JST 基準） |
| assignee_user_id | UUID | FK(users.id), NULL | 担当者ユーザーID |
| created_by_user_id | UUID | FK(users.id), NOT NULL | 作成者ユーザーID |
| created_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 作成日時 |
| updated_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 更新日時 |

> `parent_task_id` が NULL でないレコードがサブタスク。サブタスクには `parent_task_id` を設定できないよう、アプリケーション層で制御する（2階層制限）。

### notification_logs

| カラム名 | 型 | 制約 | 説明 |
|---------|---|------|------|
| id | UUID | PK, NOT NULL | ログID |
| task_id | UUID | FK(tasks.id), NOT NULL | 通知対象タスクID |
| user_id | UUID | FK(users.id), NOT NULL | 通知先ユーザーID |
| sent_at | DATE | NOT NULL | 送信日（同日重複送信防止のために使用） |
| created_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | レコード作成日時 |
| UNIQUE | — | (task_id, user_id, sent_at) | 同日重複通知防止 |
