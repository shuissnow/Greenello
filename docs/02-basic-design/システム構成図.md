# システム構成図

## 全体構成

```
[ブラウザ]
    │ HTTPS
    ▼
[Nginx リバースプロキシ]
    ├── /          → [React + Vite SPA (静的ファイル配信)]
    └── /api/v1/*  → [C# .NET Web API]
                          │
                          ├── [PostgreSQL]
                          └── [SendGrid (外部メールサービス)]

[BackgroundJob (Hosted Service)]
    └── 毎日 08:00 JST に期限切れタスクを検出・メール送信
```

## コンポーネント一覧

| コンポーネント | 役割 | 技術 |
|-------------|------|------|
| フロントエンド | SPA によるユーザーインターフェース | React 18 + Vite + TypeScript |
| バックエンド | RESTful API サーバー・ビジネスロジック・セッション管理 | C# .NET 8 Web API |
| データベース | データ永続化 | PostgreSQL 16 |
| リバースプロキシ | ルーティング・静的ファイル配信 | Nginx |
| バックグラウンドジョブ | 期限切れタスク検出・メール送信（バックエンドプロセス内） | .NET Hosted Service |
| メール配信 | 期限切れ通知メール送信 | SendGrid (外部サービス) |

## 通信経路

| 送信元 | 送信先 | プロトコル | 概要 |
|-------|-------|----------|------|
| ブラウザ | Nginx | HTTPS (443) | ユーザーリクエスト全般 |
| Nginx | フロントエンド (静的ファイル) | 内部 | HTML/JS/CSS 配信 |
| Nginx | バックエンド | HTTP (内部) | API リクエストのプロキシ |
| バックエンド | PostgreSQL | TCP (5432) | データ読み書き |
| バックエンド | SendGrid | HTTPS | メール送信 |
| フロントエンド (ブラウザ) | バックエンド | HTTPS (`/api/v1/*`) | API 呼び出し（Cookie セッション） |
