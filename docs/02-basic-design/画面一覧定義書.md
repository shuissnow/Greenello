# 画面一覧定義書

## 画面一覧

| 画面ID | 画面名 | URL | 概要 | 認証要否 |
|-------|-------|-----|------|---------|
| S-001 | 新規登録画面 | `/register` | メールアドレスとパスワードで新規アカウントを作成する | 不要（未ログイン時のみアクセス可） |
| S-002 | ログイン画面 | `/login` | メールアドレスとパスワードでログインする | 不要（未ログイン時のみアクセス可） |
| S-003 | ホーム画面 | `/` | 参加中のボード一覧を表示する。ボードの新規作成が可能 | 必要 |
| S-004 | ボード画面（カンバン表示） | `/boards/:boardId` | ステータス別カラムにタスクカードを表示する。デフォルト表示 | 必要（ボードメンバーのみ） |
| S-005 | ボード画面（リスト表示） | `/boards/:boardId?view=list` | タスクを一覧テーブルで表示する | 必要（ボードメンバーのみ） |
| S-006 | ボード画面（カレンダー表示） | `/boards/:boardId?view=calendar` | 期限日をカレンダー上にタスクとして表示する | 必要（ボードメンバーのみ） |
| S-007 | ボード画面（進捗サマリー表示） | `/boards/:boardId?view=summary` | タスクの進捗状況をグラフ・集計で可視化する | 必要（ボードメンバーのみ） |
| S-008 | タスク詳細モーダル | （各ボード画面内でオーバーレイ表示） | タスクの作成・編集・削除・サブタスク管理・担当者アサインを行う | 必要（ボードメンバーのみ） |
| S-009 | ボード設定画面 | `/boards/:boardId/settings` | ボード削除を行う | 必要（ボードオーナーのみ） |
| S-010 | アカウント設定画面 | `/settings/account` | パスワード変更を行う | 必要 |

## 各画面の主要コンポーネント

### S-001 新規登録画面
- メールアドレス入力フィールド
- パスワード入力フィールド（マスク表示）
- 「登録」ボタン
- ログイン画面へのリンク
- バリデーションエラー表示エリア

### S-002 ログイン画面
- メールアドレス入力フィールド
- パスワード入力フィールド（マスク表示）
- 「ログイン」ボタン
- 新規登録画面へのリンク
- バリデーション・ロックエラー表示エリア

### S-003 ホーム画面
- グローバルナビゲーション（ログアウト・アカウント設定へのリンク）
- ボードカード一覧
- 「新規ボード作成」ボタン

### S-004〜S-007 ボード画面（共通）
- グローバルナビゲーション
- 表示切替タブ（カンバン / リスト / カレンダー / 進捗サマリー）
- 検索バー・フィルターパネル
- 「タスク追加」ボタン
- CSV エクスポートボタン
- ボード設定へのリンク（オーナーのみ表示）

### S-008 タスク詳細モーダル
- タイトル入力フィールド
- ステータス選択（Inbox / 進行中 / 完了）
- 優先度選択（高 / 中 / 低）
- カテゴリ入力フィールド
- 期限入力（カレンダーピッカー）
- 担当者選択（ボードメンバーリスト）
- サブタスクリスト・追加フォーム（親タスクのみ表示）
- 「保存」ボタン・「削除」ボタン

### S-009 ボード設定画面
- 「ボードを削除」ボタン（確認ダイアログ付き）

### S-010 アカウント設定画面
- 現在のパスワード入力フィールド
- 新しいパスワード入力フィールド
- 新しいパスワード（確認）入力フィールド
- 「変更」ボタン
- バリデーションエラー表示エリア
